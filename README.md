# spaceflamingo

Nopea Three.js + TypeScript -pohja selaimessa pyörivälle 3D-pelille.

🎮 **Play online:** https://juhku1.github.io/spaceflamingo/

## Vaatimukset

- Node.js 20+

## Käynnistys

1. Asenna riippuvuudet: `npm install`
2. Käynnistä kehityspalvelin: `npm run dev`
3. Avaa selaimessa Viten URL (oletus: http://localhost:5173)

## Build

- Tuotantobuild: `npm run build`
- Esikatselu lokaalisti: `npm run preview`

## Ohjaus / Controls

- WASD + hiiri / mouse: liike ja kameran suunta / move and look around
- Shift: juoksu / run
- Space: hyppy / jump
- Hiiren vasen / Left mouse: ammu / shoot
- Hiiren oikea (pohjassa) / Right mouse (hold): tähtää / aim
- H: avaa/sulje ohjeikkuna / open-close help
